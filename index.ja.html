機能

Json

RFC 8259 に基づく pull 型の JSON パーザーのライブラリー。

Lattice

ビタビアルゴリズムと A*（エースター） アルゴリズムの実装です。

Property

プラットフォームに依存しない、アプリケーション設定の読み書き処理。

Text

Unicode 14.0.0 に準拠した書記素分割器、および文字列エンコーディング変換器。

Trie

共通接頭辞検索を行うことができるデータ構造で、ダブル配列で実装しています。
